<h2> </h2>

Rustt (读音 / rʌsˈti /) 是 RustCn 翻译计划的英文缩写，负责将国外优秀的技术文章、学习教程、新闻资讯翻译成中文。


## 订阅

- **所有**翻译文章都会存在本仓库中，大家可在右上角，🌟 收藏
- **部分**翻译文章还会同步推送微信公众号: `studyrust`
